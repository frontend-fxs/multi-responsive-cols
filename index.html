Multiple Device Width Column System applyied to Sitefinity Proposition

This is a proposal to implement a more flexible system of columns within sitefinity that takes into account different device widths

The style sheet already has this system as shown in the small sample below:

This default system collapses to full width in devices less than 1024 pixels wide

fxs_col_67
fxs_col_33

This "st" system collapses to full width in devices less than 768 pixels wide

fxs_col_67_st
fxs_col_33_st

This "l" system collapses to full width in devices less than 1150 pixels wide

fxs_col_67_l
fxs_col_33_l

This "xl" system collapses to full width in devices less than 1280 pixels wide

fxs_col_67_xl
fxs_col_33_xl
